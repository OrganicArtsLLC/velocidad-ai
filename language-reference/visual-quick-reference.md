## Ser vs Estar

| Use **SER** for: | Use **ESTAR** for: |
|------------------|---------------------|
| Identity | Location |
| Origin | Temporary state |
| Occupation | Ongoing action |
| Characteristics | Emotions |
| Time/Date | Appearance change |
| Religion | Result of action |

**Examples:**
- **Soy** [nombre] (identity)
- **Estoy** en Vegas (location)
- **Soy** programador (occupation)
- **Estoy** cansado (temporary state)

## The 3 Stress Rules

| Rule | When | Example |
|------|------|---------|
| **1** | Ends in vowel, n, or s | Stress second-to-last | c**a**sa, hab**lan** |
| **2** | Ends in other consonant | Stress last syllable | hab**lar**, espa**ñol** |
| **3** | Has accent mark (á é í ó ú) | Stress that syllable | ca**fé**, lá**piz** |

## Your Friction Points (Track These)

| What broke | Correct form | Why |
|------------|--------------|-----|
| muchos gracias | muchas gracias | gracias is feminine plural |
| para tu | para ti / para usted | after preposition, use ti (object form) |
| **une café** | **un café** | une doesn’t exist — German *eine* bleeding in |
| **grande café** | **café grande** | adjective after noun in Spanish |
| **queiro** | **quiero** | letters are ie not ei (e→ie stem change) |
| **etudiante habla** | **Estoy aprendiendo** | wrong construction; use estar + gerund |
| **no algo más** | **Nada más** | can’t negate this way; nada más IS the answer |
