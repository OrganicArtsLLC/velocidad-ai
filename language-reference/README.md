The library is organized into four sections. Each section serves a different purpose. Start wherever your need is sharpest.

| Section | Purpose | When to use |

## How to Use This Library

**Before a session:** Scan the vocabulary or patterns for today's world.
**During a session:** Look up a word, verb form, or preposition in seconds.
**After friction:** Find the relevant reference, understand why it broke, build a cloze card.
**Weekly:** Read one new methods file or system layer. Expand your toolkit.

This is a living library. Add your own notes, examples, and observations as you learn. The best reference is one you've written in.
