# Layer 1: Sound System

**Goal:** Pronounce Spanish clearly enough to be understood.

---

## The 5 Vowels (Never Change)

Unlike English (chaos) and German (ä/ö/ü variations), Spanish vowels are **always the same**:

| Vowel | Sound | Like German | Example |
|-------|-------|-------------|---------|
| **a** | "ah" | *Mann* | c**a**sa, gr**a**cias |
| **e** | "eh" | *Bett* | caf**é**, l**e**che |
| **i** | "ee" | *Lied* | s**í**, am**i**go |
| **o** | "oh" | *Boot* | n**o**, c**o**mo |
| **u** | "oo" | *gut* | t**ú**, m**u**cho |

**Rule:** What you see is what you say. Always.

---

## Consonants (Mostly Familiar)

### Easy Ones (Same as German/English)
- **b, d, f, k, l, m, n, p, s, t** — exactly as you expect

### Watch These:

#### **c** — Depends on next letter
- Before **e/i**: soft "s" sound → *gra**c**ias* (GRAH-see-ahs)
- Before **a/o/u**: hard "k" sound → ***c**asa* (KAH-sah)

🇩🇪 **German speaker:** Same as *Celsius* vs *Cola*

#### **g** — Depends on next letter
- Before **e/i**: breathy "h" sound → ***g**ente* (HEN-teh)
- Before **a/o/u**: hard "g" sound → ***g**ato* (GAH-toh)

🇩🇪 **German speaker:** Like *Geld* (soft) vs *Gott* (hard), but softer

#### **h** — Always silent
- ***h**ola* = OH-lah (not HO-lah)

🇩🇪 **German speaker:** Not like *Haus*. Pretend it's not there.

#### **j** — Breathy "h" sound
- *tra**b**ajo* = trah-BAH-hoh

🇩🇪 **German speaker:** Like *Bach* (the composer), same throat sound

#### **ll** — Like English "y"
- *lla***m**o* = YAH-moh

#### **ñ** — Like "ny" in "canyon"
- *ma**ñ**ana* = mah-NYAH-nah

🇩🇪 **German speaker:** Not in German, but easy. Think "lasagna"

#### **r** — Tap once (single r) or roll (double rr)
- Single: *pe**r**o* = light tap
- Double: *pe**rr**o* = roll it

🇩🇪 **German speaker:** Your German *r* is guttural (throat). Spanish *r* is tongue tip on roof of mouth. Practice: *tres, tren, trabajo*

#### **v** — Pronounced like "b"
- *vino* = BEE-noh (not VEE-noh)

🇩🇪 **German speaker:** NOT like German *Vogel*. It's a *b* sound.

#### **z** — Like "s"
- *lápiz* = LAH-pees

## The 3 Stress Rules (Critical!)

Spanish stress is **predictable** (unlike English):

### Rule 1: Ends in vowel, n, or s → Stress second-to-last syllable
- *ca**sa*** = KAH-sah
- *ha**blan*** = AH-blan
- *gra**cias*** = GRAH-see-ahs

### Rule 2: Ends in other consonant → Stress last syllable
- *hab**lar*** = ah-BLAR
- *es**pa**ñol*** = es-pah-NYOL

### Rule 3: Accent mark (á é í ó ú) → Stress that syllable
- *ca**fé*** = kah-FEH
- *lá**piz*** = LAH-pees
- *úl**ti**mo*** = OOL-tee-moh

🇩🇪 **German speaker:** German stress is unpredictable (*BERlin* vs *berLIN*). Spanish stress follows these 3 rules, period.

## Common Pronunciation Mistakes (Avoid These)

| ❌ English habit | ✅ Spanish way | German equivalent |
|-----------------|---------------|-------------------|
| "grassy-us" | GRA**H**-see-ahs | Like *Mann* + *Lied* |
| "kah-fay" | kah-**FEH** | Sharp *e*, not diphthong |
| "pore fah-vor" | pohr fah-**BOHR** | *Boot*, not *vor* |
| "moo-cho" | **MOO**-choh | Stress first syllable |

## Practice Sentences (Say Aloud)

1. **Hola, buenos días** — OH-lah, BWEH-nohs DEE-ahs
2. **Muchas gracias** — MOO-chahs GRAH-see-ahs
3. **No necesito cambio** — noh neh-seh-SEE-toh KAHM-bee-oh
4. **Quiero un café** — kee-EH-roh oon kah-FEH
5. **Para usted** — PAH-rah oo-STED
