# Cognate Accelerator — Unlock 3000+ Words From English

English and Spanish share thousands of words through Latin, French, and Greek ancestry. If you speak English, you already know more Spanish than you think. This guide maps the patterns so you can produce Spanish words you've never formally studied.

**The deal:** Learn 18 suffix rules and you can convert thousands of English words into correct (or nearly correct) Spanish on the fly. This is the single highest-ROI hour you'll spend.

**The risk:** Some conversions produce false friends — words that look like English but mean something different. The [false friends section](#false-friends--words-that-will-betray-you) at the bottom covers the dangerous ones.

## The 18 Suffix Conversion Rules

### 1. -tion → -ción (feminine)

The biggest single pattern. Almost every English -tion word converts.

|---------|---------|-------|
| nation | na**ción** | |
| education | educa**ción** | |
| information | informa**ción** | |
| situation | situa**ción** | |
| communication | comunica**ción** | |
| operation | opera**ción** | |

| conversation | conversa**ción** | |
| tradition | tradi**ción** | |
| direction | direc**ción** | |
| action | ac**ción** | |
| construction | construc**ción** | |
| protection | protec**ción** | |
| preparation | prepara**ción** | |
| organization | organiza**ción** | |

| concentration | concentra**ción** | |
| reservation | reserva**ción** | |
| application | aplica**ción** | |
| production | produc**ción** | |

**All -ción words are feminine.** *La información*, *la situación*. No exceptions.

**Stress:** Always on the final syllable: `na-CIÓN`, `si-tua-CIÓN`.

## False Friends — Words That Will Betray You

## Production Strategy

When you're in a conversation and need a word you don't know:

1. **Think of the English word**
2. **Apply the suffix rule** (-tion → -ción, -ty → -dad, etc.)
3. **Speak it with Spanish pronunciation** (every vowel pronounced, stress on right syllable)
4. **Watch the listener's face** — if they don't understand, try a different word

This works **70-80% of the time** for polysyllabic English words of Latin origin. It fails for:
- Short Anglo-Saxon words (house, dog, food, work)
- False friends (see above)
- Words with no Latin root

**The cognate radar:** If an English word has more than two syllables and sounds "formal" or "Latin-y," try converting it. *Communication* → *comunicación*. *Responsibility* → *responsabilidad*. *Incredible* → *increíble*.
