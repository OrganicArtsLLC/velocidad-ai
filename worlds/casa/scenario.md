# World: Casa (House Workers)

Communicating with workers helping on the house. Directions, logistics, appreciation.

---

## Why This World Matters

- Real stakes: miscommunication = wasted time or wrong work
- Practical vocabulary: tools, rooms, directions, materials
- Builds respect and connection
- Recurring opportunity (project-based)

## Goals

- Give basic directions clearly
- Ask if something is done / how long
- Express appreciation genuinely
- Handle a question about the work
- Discuss a change or problem

## Scenario Ladder

### L1: Basic Directions (30 sec)

```
You:     Hola, buenos días. ¿Cómo está?
Worker:  Bien, gracias. ¿Y usted?
You:     Bien, gracias. Esto va aquí, por favor. [pointing]
Worker:  Está bien.
You:     Gracias.
```

### L2: Locations + Specifics (1-2 min)

- En la cocina / en el baño / en el garaje / afuera / arriba / abajo
- A la derecha / a la izquierda / al lado de / enfrente de
- Más arriba / más abajo / un poco más
- ¿Puede mover esto aquí?

### L3: Questions + Logistics (2-3 min)

- ¿Cuánto tiempo va a tomar?
- ¿Necesita algo? / ¿Necesita más material?
- ¿Ya terminaron con [area]?
- ¿Hay algún problema?
- Voy a estar aquí / Voy a salir un rato.

### L4: Discussion + Problem Solving (3-5 min)

- Explain what you want changed
- Ask about options: "¿Qué recomienda?"
- Discuss timing: "¿Puede venir mañana?"
- Express appreciation: "Están haciendo un trabajo excelente."

### L5: Full Coordination

- Plan a project timeline
- Discuss materials and costs
- Handle an unexpected issue
- Negotiate schedule changes

## Essential Vocabulary

### Rooms
- la cocina (kitchen), el baño (bathroom), la sala (living room)
- el cuarto / la habitación (bedroom), el garaje (garage)
- el patio / el jardín (yard), el techo (roof)

### Directions
- aquí (here), allá (there), arriba (up), abajo (down)
- a la derecha (right), a la izquierda (left)
- adentro (inside), afuera (outside)
- al lado de (next to), enfrente de (in front of)

### Common Actions
- pintar (paint), limpiar (clean), mover (move)
- instalar (install), reparar (repair), cortar (cut)
- quitar (remove), poner (put/place)

### Time
- hoy (today), mañana (tomorrow), esta semana (this week)
- ¿Cuánto tiempo? (how long?), una hora (an hour)
- Ya está / ya terminé (it's done / I'm done)

### Appreciation
- Gracias por su trabajo.
- Quedó muy bien.
- Excelente trabajo.
- Se lo agradezco mucho.
